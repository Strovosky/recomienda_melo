# Project Name
A Django web application for creating posts about interesting places in the city and reviewing them. Here we also practice creating a User and loging them in.
We also have pagination. We can like posts (each post only one), etc.
The project doesn't have an API and has some functions created as function views and class views with the same purpose in mind just for the fun of
seeing how to create the funtion both ways.

## Setup
1. Clone repo
2. Create virtual environment
3. Install dependencies: `pip install -r requirements.txt`
4. Run migrations: `python manage.py migrate`
5. Start server: `python manage.py runserver`
